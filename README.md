# Movie-recommendation-system
This Machine learning project recommends movies based on the movie user provides.It uses Natural languages processing to find similarity between movies.
Features
1.Recommends movies based on text similarity
2.Uses TF-IDF Vectorization
3.Measures similarity using Cosine Similarity
4.Can be integrated with Streamlit for a web UI
